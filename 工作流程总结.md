# 2025 年华南地区新能源双碳目标工作报告 - 工作流程总结

## 项目概述
本项目完成了 2025 年华南地区（广东、广西、海南）新能源双碳目标工作报告的全流程生成，包括数据整理、可视化图表制作和 Word 报告文档生成。

---

## 工作流程

### 第一步：项目结构创建
```
📁 2025-新能源华南双碳报告/
├── 📁 data/          # 数据文件
├── 📁 charts/        # 可视化图表
├── 📁 reports/       # 最终报告文档
├── generate_data.py  # 数据生成脚本
└── generate_report.py # 报告生成脚本
```

### 第二步：数据生成与 Excel 创建
**文件：** `generate_data.py`

**功能：**
- 生成 2025 年华南三省区月度新能源发电量数据（单位：亿千瓦时）
- 创建格式化的 Excel 文件，包含：
  - 月度数据表格（带表头样式、边框、对齐）
  - 全年合计汇总行
  - 各省区及总计数据

**输出：** `data/2025 年华南新能源发电量.xlsx`

### 第三步：数据可视化
**功能：**
使用 Python matplotlib 生成 3 种图表：

1. **月度发电量折线图** (`月度发电量折线图.png`)
   - 展示广东、广西、海南及总计的全年月度趋势
   - 标注数据点数值
   - 清晰显示季节性变化特征

2. **全年发电量占比饼图** (`全年发电量占比饼图.png`)
   - 展示三省区全年发电量占比
   - 广东省：60.8%
   - 广西壮族自治区：24.5%
   - 海南省：14.7%

3. **各省月度对比柱状图** (`各省月度对比柱状图.png`)
   - 并列柱状图对比三省区每月发电量
   - 直观显示各省贡献差异

### 第四步：Word 报告生成
**文件：** `generate_report.py`

**功能：**
使用 python-docx 库生成专业 Word 报告，包含：

**文档结构：**
1. 封面（标题、副标题、日期、编制单位）
2. 目录
3. 一、报告摘要
   - 关键指标完成情况表格
4. 二、华南地区新能源发展背景
   - 资源优势说明
5. 三、2025 年度发电量数据分析
   - 月度发电量趋势分析
   - 月度数据统计表
   - 各省区贡献占比
6. 四、双碳目标完成情况
   - 主要成就列表
7. 五、存在问题与挑战
8. 六、2026 年工作建议
9. 七、结语

**格式特点：**
- 中文字体支持（微软雅黑、黑体、楷体）
- 专业表格样式
- 统一的段落格式和行距
- 分页和章节结构

**输出：** `reports/2025 年华南地区新能源双碳目标工作报告.docx`
