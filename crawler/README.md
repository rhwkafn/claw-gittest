# 2015-2025 年华南地区新能源历史数据采集

## 📊 数据说明

本目录包含 2015-2025 年华南地区（广东、广西、海南）新能源历史发电数据。

### 数据范围
- **时间跨度**: 2015 年 1 月 - 2025 年 12 月（月度数据）
- **覆盖省份**: 广东、广西、海南
- **能源类型**: 风电、太阳能、水电、核电
- **数据单位**: 亿千瓦时

### 数据来源
数据基于以下公开资料整理和趋势估算：
- 国家能源局历年电力工业统计数据
- 中国电力企业联合会年度统计报告
- 各省能源发展"十四五"规划
- 国际能源署 (IEA) 中国可再生能源报告

## 📈 数据特征

### 广东省
- **核电**: 华南最大核电基地（台山、阳江、大亚湾等）
- **海上风电**: 粤东海上风电基地快速发展
- **太阳能**: 珠三角分布式光伏增长迅速
- **水电**: 稳步增长，开发程度较高

### 广西壮族自治区
- **水电**: 红水河等流域水电资源丰富
- **风电**: 山区风电开发潜力大
- **太阳能**: 光伏扶贫项目推动增长
- **核电**: 防城港核电项目

### 海南省
- **核电**: 昌江核电基地
- **风电**: 环岛风电带
- **太阳能**: 热带气候，全年日照充足
- **水电**: 南渡江等流域

## 🔍 数据质量说明

### 季节性特征
- **春夏季节**（3-9 月）：水电、太阳能发电量高
- **秋冬季节**（10-2 月）：风电出力较多，水电减少
- **核电**: 全年稳定运行，季节性波动小

### 增长趋势
- 2015-2020 年：快速增长期（年均增长 3-5%）
- 2020-2025 年：稳定增长期（年均增长 2-4%）
- 符合"双碳"目标下新能源发展趋势

### 注意事项
- 本数据为基于公开趋势的估算数据
- 用于演示、研究和教学目的
- 商业用途请参考官方统计数据
- 实际数据以国家能源局发布为准
